# Automower Connect OAuth Demo

This is a Node.js app that authenticates with the Husqvarna Automower Connect API using OAuth2.

## Setup

1. Copy `.env.example` to `.env` and fill in your values from the Husqvarna Developer Portal.
2. Install dependencies:
   ```
   npm install
   ```
3. Run locally:
   ```
   node server.js
   ```
4. Deploy to [Render.com](https://render.com) and set the same environment variables there.
